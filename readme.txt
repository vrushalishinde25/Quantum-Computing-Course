Files:
1) Qiskit_Tutorial_Phase_1 Final.ipnyb - this is a tutorial notebook file for phase 1
2) Quantum computing with Qiskit - Phase 1.pptx - slides for phase 1
3) Qiskit_Tutorial_Phase_2 Final.ipnyb - this is a tutorial notebook file for phase 2
4) Quantum computing with Qiskit - Phase 2.pptx - slides for phase 2


How to run the notebook:
1) Install Anaconda from https://www.anaconda.com/
2) Install Python 3.x (https://www.python.org/downloads/)
3) Install Qiskit (version 0.42.0) using:
	 pip install qiskit[visualization]

How to run the .py file:
1) Any editor for python file e.g. VS Code.
2) Install Python 3.x (https://www.python.org/downloads/)
3) Install all the needed dependencies from the requirements.txt file using:
	pip install -r requirements.txt
